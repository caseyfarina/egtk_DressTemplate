  Dress to Impress вҖ” EGTK 2D Template Plan

  What the game is

  A 2D sprite-based dress-up game. A character stands center-screen with empty
  outfit slots. The player drags (or clicks) clothing items from a side panel
  onto the character. A theme prompt ("Beach Party!", "Black Tie Gala") is
  revealed at the start. When time runs out, an OutfitScorer evaluates how well
  the outfit matches the theme and fires events that trigger judge dialogue and
  score display.

  ---
  Key design decisions

  1. Drag vs click to equip: InputSpriteDrag lets students drag items to
  character slots. Slot detects the drop via OnCollisionEnter2D /
  OnTriggerEnter2D (trigger zones on each slot). Simple drop detection, no
  complex inventory system.
  2. Unequip: Clicking an equipped slot fires DressUpSlot.Unequip() via
  InputSpriteClick + UnityEvent wiring. Item returns to panel (or just reappears
   вҖ” students can wire ActionAnimateTransform for snap-back).
  3. Score calculation: OutfitScorer.CalculateScore() is called by wiring
  GameTimerManager.onTimerEnd вҶ’ OutfitScorer.CalculateScore(). Score result
  wires to GameCollectionManager.SetValue(int).
  4. Theme matching: Each ClothingItem has a themeTags string list. OutfitScorer
   counts how many equipped items share a tag with the current theme. Bonus
  points per matching item.
  5. No character controller needed: This is a point-and-click/drag game вҖ” no
  CharacterControllerCC required.

  2. ClothingItem + DressUpSlot (core dress-up logic)
  3. OutfitScorer + DressUpThemeManager (game loop logic)
  4. Wire the template scene with placeholder sprites
  5. Student guide / quick-start doc
